
# Developer Connector

A simple application to connect developers.

## Description

Application uses JWT auth. Allows users to create profiles, posts & comments.

### API Dev Stack
* Node JS
* Mongo DB
* Express
* Mongoose

### Client Dev Stack (located inside /client)
* React JS
* Bootstrap CSS
* Redux

## Getting Started

### Dependencies

* Node Package Manager

### Installing

* clone repo
```
npm i
```
```
npm run dev
```
